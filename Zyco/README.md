# Zyco-editor
```$xslt
It is a text-editor
```

## Features:
- Upload and save file
- Edit file
- Save your progress
- Terminal
- Draw on the go
- Insert pictures
- Search (Case insensitive)
- Keyboard and Mouse binding functionality
- Make text Bold, italic, or underline text
- Change text to different font styles.
